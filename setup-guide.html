1

建立 Google Cloud 專案

前往 Google Cloud Console

console.cloud.google.com
前往 https://console.cloud.google.com 並用學校的 Google 帳號登入。

建立新專案

  • 點擊頁面頂端的專案選擇器（通常顯示「選取專案」或目前專案名稱）
  • 在彈出視窗右上角點擊 新增專案
  • 專案名稱輸入：Sjps-AdPush
  • 點擊 建立，等待幾秒
  • 建立完成後，確認右上角顯示的是 Sjps-AdPush 專案
2

啟用 Google Drive API

  • 在左側選單點擊 API 和服務 → 程式庫
  • 在搜尋框輸入：Google Drive API
  • 點擊搜尋結果中的 Google Drive API（Google 官方出品）
  • 點擊藍色的 啟用 按鈕
  • 等待幾秒，頁面跳轉後顯示「已啟用」即完成
✅ 完成後，你會看到 Google Drive API 的管理頁面，狀態顯示「已啟用」

OAuth 總覽

ℹ 尚未針對這項專案設定 OAuth 用戶端。 建立 OAuth 用戶端

步驟 A：點擊左側「品牌」設定

  • 在左側選單點擊 品牌（Brand）
  • 應用程式名稱 填入：SJPS AdPush
  • 使用者支援電子郵件：選擇你的 Google 帳號
  • 開發人員聯絡資訊：填入你的 Email
  • 點擊 儲存並繼續

步驟 B：目標對象設定

  • 點擊左側 目標對象
  • 使用者類型選擇 外部（如果只有學校帳號也可選「內部」）
  • 點擊 建立
  • 在測試使用者中，點擊「新增使用者」，加入你自己的 Google 帳號 Email
  • 點擊 儲存
因為這個應用是學校內部使用，不需要通過 Google 官方審核，維持「測試」狀態即可。
4

建立 API 金鑰（用來讀取 Drive 檔案）

  • 左側選單點擊 用戶端（Clients）
  • 點擊上方 ＋ 建立用戶端
  • 應用程式類型選擇：API 金鑰
  • 點擊 建立，會出現一串 AIzaSy... 開頭的金鑰
  • 點擊 複製 並暫存在記事本

限制 API 金鑰（建議）安全性

  • 點擊剛建立的 API 金鑰名稱，進入編輯
  • 在「應用程式限制」選擇 HTTP 參照網址（網站）
  • 加入：https://issac-sjps.github.io/*
  • 在「API 限制」選擇 限制金鑰 → 勾選 Google Drive API
  • 點擊 儲存
5

建立 OAuth 2.0 用戶端 ID（用來上傳檔案）

  • 一樣在 用戶端 頁面，點擊 ＋ 建立用戶端
  • 應用程式類型選擇：網頁應用程式
  • 名稱填入：AdPush Web

填入授權來源 重要

建立 OAuth 2.0 用戶端 ID
應用程式類型
網頁應用程式

已授權的 JavaScript 來源

https://issac-sjps.github.io

← 填入這個網址，不要加斜線結尾

已授權的重新導向 URI（可留空）

（不需填寫）
  • 在「已授權的 JavaScript 來源」點擊「新增 URI」
  • 填入：https://issac-sjps.github.io（注意：結尾不加斜線）
  • 點擊 建立
  • 畫面出現 用戶端 ID（xxx.apps.googleusercontent.com）和 用戶端密碼
  • 複製 用戶端 ID（不是密碼！）並暫存在記事本
6

設定 Google Drive 資料夾公開共用

⚠️ 這步非常重要！若資料夾不是公開，API 金鑰無法讀取檔案清單。
  • 打開 Google Drive，找到你的廣告媒體資料夾
  • 對資料夾按右鍵 → 共用（或點選資料夾後點上方「共用」按鈕）
  • 在「一般存取」區塊，點擊下拉選單，從「受限」改為 知道連結的任何人
  • 右側角色選擇 檢視者（只需要讀取，不需要編輯）
  • 點擊 完成

取得資料夾 ID

drive.google.com/drive/folders/18_RG5qrn-8WVxyq6QwyNIW3afIMtKZGi
網址中 folders/ 後面那一段就是資料夾 ID
你的 ID 是：18_RG5qrn-8WVxyq6QwyNIW3afIMtKZGi
7

填入 AdPush 管理面板

  • 打開你的網站 https://issac-sjps.github.io/Sjps-AdPush/
  • 按鍵盤 A 開啟管理面板
  • 在「API 金鑰」填入 Step 4 取得的 AIzaSy...
  • 在「Client ID」填入 Step 5 取得的 xxx.apps.googleusercontent.com
  • 在「資料夾 ID」填入 18_RG5qrn-8WVxyq6QwyNIW3afIMtKZGi
  • 「每張秒數」填入希望每張投影片停留的秒數（預設 8 秒）
  • 點擊 💾 儲存設定，系統自動重新載入 Drive 中的媒體

上傳媒體

  • 在管理面板中點擊 🔑 使用 Google 登入，授權帳號
  • 點擊上傳區塊或拖曳檔案（PNG / JPG / PDF / PPT / PPTX / MP4）
  • 上傳完成後系統自動重新整理，檔案立刻加入輪播
也可以直接到 Google Drive 資料夾手動上傳，然後在管理面板點「↺ 重新載入」。

📋 你需要準備的資料清單

項目在哪裡取得格式
API 金鑰 Google Cloud → 用戶端 → 建立 API 金鑰 AIzaSy...
OAuth Client ID Google Cloud → 用戶端 → 建立 OAuth 2.0 用戶端 ID xxx.apps.googleusercontent.com
資料夾 ID Google Drive 資料夾 URL 中 folders/ 後面的部分 18_RG5qrn-...

🎉 設定完成！

完成以上步驟後，你的廣告播放系統就可以正常運作了。
把媒體檔案丟進 Drive 資料夾，螢幕上就會自動輪播。
